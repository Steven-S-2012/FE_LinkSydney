**********************************************************************
This is the instruction to show how to install this react package.
**********************************************************************
This file contains whole develop environment which wrap by the Webpack.
So please follow the steps below for the quick installation.
1) Instal Node.js from website (https://nodejs.org/en).
2) Unzip the file.
3) Run the VScode.In the Integrated Terminal, "cd" to the file directory.
4) Run "npm install".
5) Run "npm run dev".
6) Enter "Localhost:3000" in the browser.
